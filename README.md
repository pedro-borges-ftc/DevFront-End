# DevFront-End

Repositório de estudos da disciplina de Desenvolvimento Front-End, organizado por aulas, revisões e exercícios práticos com HTML, CSS e JavaScript.

Este README documenta apenas o conteúdo principal do repositório e **não considera a pasta `MeuPrimeiroProjeto`**.

## Visão Geral

O projeto reúne exemplos introdutórios e aplicações menores desenvolvidas ao longo das aulas, cobrindo desde fundamentos da web até interações com DOM, consumo de API e uso de `localStorage`.

Entre os temas presentes no repositório estão:

- estruturação de páginas com HTML;
- estilização com CSS;
- responsividade e grid layout;
- manipulação do DOM com JavaScript;
- estruturas de decisão, repetição e arrays;
- formulários e tabelas;
- integração com API de CEP via `fetch`;
- exercícios de IMC e classificação de triângulos;
- exemplos de CRUD com `localStorage`;
- miniaplicações temáticas, como página de Pet Shop e projeto de gestão financeira.

## Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript
- Bootstrap em alguns exemplos
- `fetch` para consumo de API
- `localStorage` em exercícios de persistência
- Node.js com dependência `prompt-sync` para alguns testes/scripts

## Estrutura do Repositório

### `Aula00_RevisaoAlgoritmos`

Material de revisão de lógica de programação e algoritmos, com exercícios sobre:

- introdução e revisão;
- estruturas de decisão;
- estruturas de repetição;
- arrays;
- JSON e `localStorage`;
- projeto de gestão financeira (`06_ProjetoGefin`).

### `Aula01`

Conteúdo introdutório com páginas simples e exercícios das primeiras semanas:

- currículos em HTML;
- primeiros exemplos com CSS;
- exemplos iniciais com JavaScript;
- formulários, tabelas e manipulação básica do DOM.

### `Aula02`

Exercícios focados em estilização e layout:

- exemplos de CSS;
- páginas com organização visual;
- grid layout;
- site responsivo com imagens e folhas de estilo separadas.

### `Aula04`

Exemplo de página HTML com folha de estilo dedicada.

### `Aula05`

Conteúdo mais aplicado, incluindo:

- exemplos com Bootstrap;
- páginas demonstrativas;
- projeto de Pet Shop;
- consulta de CEP com integração à API ViaCEP.

### `Aula06`

Exercícios com JavaScript voltados à lógica aplicada em interfaces:

- calculadora de IMC;
- classificação de IMC;
- classificação de triângulos.

### Arquivos na raiz

- `index.html`: página inicial do repositório/disciplina.
- `package.json`: dependência Node utilizada no ambiente.

## Como Executar

Como a maior parte do conteúdo é composta por arquivos estáticos, a forma mais simples de executar é abrir os arquivos `.html` diretamente no navegador.

Exemplos:

- `index.html`
- `Aula05/PetShop/index.html`
- `Aula05/CEP/cep.html`
- `Aula06/Calculadora_IMC/calculadora_imc.html`

Se preferir, também é possível usar uma extensão como Live Server no VS Code para navegar pelas páginas localmente.

## Observações

- A pasta `node_modules` existe por causa da dependência `prompt-sync`.
- O repositório possui vários exercícios independentes, então nem todos os arquivos fazem parte de uma única aplicação integrada.
- A pasta `MeuPrimeiroProjeto` foi propositalmente deixada fora desta documentação.

## Objetivo do Repositório

Servir como base de estudo, prática e evolução em Desenvolvimento Front-End, reunindo exemplos progressivos que ajudam no aprendizado de estrutura, estilo, lógica e interação em aplicações web.
